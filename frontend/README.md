# PrivaVote Frontend

The frontend for **VeilVote**, built with **React** and **Vite**. It provides the user interface for creating proposals, participating in private voting, and interacting with the deployed Soroban smart contract.

## Project Structure

```text
frontend/
├── public/
│   └── circuits/
│       ├── circuit_final.zkey
│       └── vote.wasm
├── src/
├── .env.example
├── package.json
└── README.md
```

---

# Prerequisites

Install the following before running the application:

* Node.js (v18 or newer recommended)
* npm

Verify the installation:

```bash
node --version
npm --version
```

---

# Install Dependencies

From the `frontend` directory:

```bash
npm install
```

---

# Configure Environment Variables

Create a `.env` file by copying the provided example:

```bash
cp .env.example .env
```

Update the values in `.env`:

| Variable                  | Description                                                    |
| ------------------------- | -------------------------------------------------------------- |
| `VITE_CONTRACT_ID`        | Deployed Soroban contract ID                                   |
| `VITE_SOROBAN_RPC_URL`    | Soroban RPC endpoint                                           |
| `VITE_NETWORK_PASSPHRASE` | Stellar network passphrase                                     |
| `VITE_KEYHOLDERS`         | Comma-separated Stellar public keys of the election keyholders |

Example:

```env
VITE_CONTRACT_ID=<DEPLOYED_CONTRACT_ID>

VITE_SOROBAN_RPC_URL=https://soroban-testnet.stellar.org

VITE_NETWORK_PASSPHRASE=Test SDF Network ; September 2015

VITE_KEYHOLDERS=G...,G...,G...
```

---

# Copy Circuit Artifacts

The frontend requires the proving artifacts generated by the Circom project.

After completing the build steps in the `circuits` project, copy the following files:

```text
circuits/build/vote_js/vote.wasm
circuits/zkey/circuit_final.zkey
```

into

```text
frontend/public/circuits/
```

The directory should contain:

```text
public/
└── circuits/
    ├── vote.wasm
    └── circuit_final.zkey
```

These files are loaded by the browser to generate zero-knowledge proofs during voting.

---

# Start the Development Server

Run the Vite development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
